## Chatter

A chat based application inspired from whatsapp made in Flutter using dart and Firebase. New functionalities for calling and group chats will be further added in future.

### Functionalities

- OTP verification of phone number using firebase auth with phone number and country code
- get all user contacts of current user matching with firebase contacts for chatting
- ability to create users with username and profile pics
- ability to send images, gifs, videos, emojis and text messages to anyone
- show online/offline based on application state
- ability to reply to other user's messages
- create status like whatsapp status of any image from gallery
- ability to see status of anyone in your contact list and upload multiple statuses
